1. 建立 Expo App、Python API 和 Supabase migration 的最小目录结构。
   - 验证：App、API 和数据库配置各自能通过基础检查。
2. 实现数据库、图片识别和市场估价接口。
   - 验证：固定响应测试覆盖识别结构、候选过滤、样本门槛和价格统计。
3. 实现登录、首页、拍照确认、资产详情和手动刷新。
   - 验证：类型检查通过，关键状态有可运行检查。
4. 接通 App、Supabase 和 Python API，并补齐环境变量示例。
   - 验证：本地启动说明完整，不提交密钥或数据源登录态。
5. 运行定向测试和最终 diff 审核。
   - 验证：后端测试与前端类型检查通过，变更仅覆盖已批准 MVP。
