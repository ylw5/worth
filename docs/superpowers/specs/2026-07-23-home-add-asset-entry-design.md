# 首页添加物品入口设计

## 目标

“添加物品”不再占用独立底部 Tab，改为“我的资产”首页标题栏右上角的图标按钮。

## 交互

- 底部导航只保留“资产”和“账号”。
- “我的资产”标题栏右侧显示平台原生添加图标。
- 图标的无障碍标签为“添加物品”。
- 点击图标进入现有录入页面。
- 录入、识别和确认流程保持不变。

## 实现范围

- 从 `NativeTabs` 中移除录入页的 Trigger，但保留现有录入路由。
- 在资产首页的 `Stack.Screen` 配置中增加 `headerRight` 按钮，跳转到录入路由。
- 不新增组件、依赖或业务状态。

## 验证

- 底部导航只显示“资产”和“账号”。
- 首页标题栏显示添加图标，且点击后进入录入页。
- TypeScript 和现有项目检查通过。
