# Worth 物品市场价值趋势设计

## 目标

在资产详情中展示某一种物品过去一段时间的参考市价波动，同时继续把今天估价作为页面最重要的信息。

趋势表示每日同款在售样本的估值，不表示真实成交价。系统不伪造接入前的历史数据。

## 产品规则

- 顶部估价卡继续突出今天估价、参考区间和样本数。
- 估价卡下方增加市场趋势卡，提供 30 天、90 天和全部三个区间。
- 趋势卡展示估价中位数曲线、四分位参考区间、区间涨跌额和涨跌比例。
- 同一用户下，相同估价搜索词的资产共享市场趋势。
- 新导入资产如果已有同搜索词行情，立即展示已有趋势；首次追踪的物品显示“行情积累中”。
- 没有成功估价的日期不补点、不插值。

## 数据来源

沿用当前估价链路：

1. 使用资产已确认的 `search_query` 搜索前三页在售商品。
2. 按商品 ID 去重。
3. 由 AI 排除配件、错误型号和关键规格不同的候选。
4. 至少有 5 个匹配样本时，计算中位数和第 25–75 百分位区间。
5. 将成功结果同时保存为资产估价记录和当日市场快照。

第一版不接卖家已售列表、第三方历史行情或定时全量爬虫。

## 行情标识

第一版直接使用规范化后的 `search_query` 作为 `market_key`：

- 去除首尾空白。
- 连续空白合并为一个空格。
- 英文字母转为小写。

用户修改估价搜索词后，资产切换到新行情序列。第一版不增加独立的商品主数据或型号映射系统。

## 数据模型

新增 `market_snapshots`：

| 字段 | 用途 |
| --- | --- |
| `id` | 快照 ID |
| `user_id` | 数据所属用户 |
| `market_key` | 规范化估价搜索词 |
| `estimated_price` | 当日估价中位数 |
| `price_low` | 第 25 百分位 |
| `price_high` | 第 75 百分位 |
| `sample_count` | 参与计算的样本数 |
| `valued_on` | 行情日期 |
| `created_at` | 实际写入时间 |
| `updated_at` | 当日重新估价时间 |

`(user_id, market_key, valued_on)` 唯一。同一天再次成功估价时更新当天快照，避免折线图出现多个同日点。

现有 `valuations` 保留为单件资产的估价记录；`market_snapshots` 负责跨资产共享的行情序列。迁移时按规范化查询词和日期，将现有成功估价记录的当天最后一条回填为市场快照。

## 写入与读取

扩展现有 `record_valuation` 数据库函数：

1. 插入资产估价记录。
2. 更新资产的最新估价。
3. 按 `market_key + valued_on` 更新或插入市场快照。

资产详情同时读取：

- 当前资产及其最近估价。
- 当前 `market_key` 的市场快照，按日期升序返回。

打开详情时，如果该行情今天没有快照，则非阻塞地触发一次现有估价流程。当天已有共享快照时不重复请求市场接口。手动“刷新价格”仍可重新估价，并更新当天快照。

自动估价失败时保留已有价格和趋势，仅显示可重试提示。样本不足时不写入市场快照。

## 详情页

### 今日估价

顶部卡片保持现有位置和最大字号，将标题改为“今日参考市价”。继续展示：

- 今日估价。
- 今日参考区间。
- 相似样本数。
- 刷新操作。

### 市场趋势

趋势卡位于今日估价卡之后、资产资料之前：

- 顶部显示“市场趋势”和 30 天／90 天／全部切换。
- 主体使用折线展示每日估价，中间值使用绿色。
- 第 25–75 百分位使用低对比度区域表达，不抢今日估价。
- 底部显示所选区间的涨跌额、涨跌比例、最高估价和最低估价。
- 只有一个点时显示该点并提示“行情积累中”；没有点时显示“暂无行情”。

具体历史估价列表保留在页面底部，作为可核对的记录，不承担趋势展示。

## 边界

- 趋势是参考市价，不是实际成交价。
- 数据只能从已有估价记录和功能上线后继续积累。
- 相同搜索词共享行情；同一型号但搜索词不同不会自动合并。
- 每日最多保留一个行情点，不提供日内分时走势。
- 第一版只在用户打开详情时补当天快照，不增加后台调度基础设施。

## 验证

- 迁移能把现有估价按搜索词和日期回填，且同日只保留最后一条。
- 同一搜索词的两个资产读取相同行情。
- 不同搜索词不会串行情。
- 同一天重复刷新只更新一个市场快照。
- 样本不足和接口失败不会写入快照。
- 30 天、90 天和全部区间筛选及涨跌计算正确。
- 零个、一个和多个行情点均能正常展示。
- 今日估价在视觉层级上仍明显高于趋势卡。

## 完成标准

- 资产详情能够展示所选时间范围内的市场估值趋势。
- 今日估价仍是详情页最突出的金额。
- 新资产可以复用同搜索词的已有行情。
- 首次追踪的物品明确显示行情正在积累。
- 页面不把挂牌样本估值描述为真实成交价。
